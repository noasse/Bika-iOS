# Cloud History Sync

This project supports an optional private cloud history sync service. It is off by default. If no server URL, token, and certificate pin are configured in the app, reading history remains local only.

Do not commit your real VPS IP address, bearer token, certificate private key, or certificate pin to this repository. Enter them in the iOS/macOS settings UI at runtime.

## Architecture

```text
iOS/macOS app
  -> https://<VPS_PUBLIC_IP>:8443
  -> nginx with a self-signed IP certificate
  -> FastAPI history service
  -> SQLite database
```

The app verifies the server certificate by SHA-256 pinning. This allows HTTPS with a public IP and no domain name.

## Server Setup

These commands assume Ubuntu 24.04 and root access. Replace every placeholder before running:

- `<VPS_PUBLIC_IP>`: your VPS public IPv4 address.
- `<YOUR_LONG_TOKEN>`: a long random token generated by `openssl rand -hex 32`.

### 1. Install Packages

```bash
apt update
apt install -y docker.io docker-compose-v2 openssl sqlite3 ufw
systemctl enable --now docker
```

### 2. Create Directories

```bash
mkdir -p /opt/bika-history-sync/app
mkdir -p /opt/bika-history-sync/data
mkdir -p /opt/bika-history-sync/certs
mkdir -p /opt/bika-history-sync/nginx
cd /opt/bika-history-sync
```

### 3. Generate Token

```bash
openssl rand -hex 32
```

Create `.env`:

```bash
cat > .env <<'EOF'
HISTORY_SYNC_TOKEN=<YOUR_LONG_TOKEN>
HISTORY_DB_PATH=/data/history.sqlite3
HISTORY_MAX_ITEMS=200
EOF
```

### 4. Generate a Self-Signed IP Certificate

Create an OpenSSL config with the VPS IP in `subjectAltName`:

```bash
cat > certs/openssl.cnf <<'EOF'
[req]
default_bits = 2048
prompt = no
default_md = sha256
distinguished_name = dn
x509_extensions = v3_req

[dn]
CN = bika-history-sync

[v3_req]
subjectAltName = @alt_names
keyUsage = critical, digitalSignature, keyEncipherment
extendedKeyUsage = serverAuth

[alt_names]
IP.1 = <VPS_PUBLIC_IP>
EOF
```

Generate the certificate:

```bash
openssl req -x509 -nodes -days 3650 \
  -newkey rsa:2048 \
  -keyout certs/history.key \
  -out certs/history.crt \
  -config certs/openssl.cnf
```

Verify the IP SAN:

```bash
openssl x509 -in certs/history.crt -noout -text | grep -A1 "Subject Alternative Name"
```

### 5. Create the API Service

```bash
cat > app/requirements.txt <<'EOF'
fastapi==0.115.6
uvicorn[standard]==0.34.0
pydantic==2.10.4
EOF
```

```bash
cat > app/main.py <<'EOF'
import hmac
import os
import sqlite3
from datetime import datetime, timezone
from typing import Optional

from fastapi import FastAPI, Header, HTTPException, Query
from pydantic import BaseModel, Field

TOKEN = os.environ["HISTORY_SYNC_TOKEN"]
DB_PATH = os.environ.get("HISTORY_DB_PATH", "/data/history.sqlite3")
MAX_ITEMS = int(os.environ.get("HISTORY_MAX_ITEMS", "200"))

app = FastAPI(title="Bika History Sync")


class HistoryItemIn(BaseModel):
    comicId: str = Field(min_length=1)
    title: str = Field(min_length=1)
    author: Optional[str] = None
    thumbPath: Optional[str] = None
    thumbServer: Optional[str] = None
    lastReadAt: str
    episodeOrder: Optional[int] = None
    episodeTitle: Optional[str] = None
    pageIndex: Optional[int] = None


class BatchIn(BaseModel):
    items: list[HistoryItemIn]


def now_iso() -> str:
    return datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")


def connect():
    conn = sqlite3.connect(DB_PATH, timeout=5)
    conn.row_factory = sqlite3.Row
    return conn


def require_auth(authorization: Optional[str]):
    expected = f"Bearer {TOKEN}"
    if not authorization or not hmac.compare_digest(authorization, expected):
        raise HTTPException(status_code=401, detail="Unauthorized")


def init_db():
    with connect() as db:
        db.execute("""
        CREATE TABLE IF NOT EXISTS history_items (
            comic_id TEXT PRIMARY KEY,
            title TEXT NOT NULL,
            author TEXT,
            thumb_path TEXT,
            thumb_server TEXT,
            last_read_at TEXT NOT NULL,
            episode_order INTEGER,
            episode_title TEXT,
            page_index INTEGER,
            created_at TEXT NOT NULL,
            updated_at TEXT NOT NULL,
            deleted_at TEXT
        )
        """)
        db.execute("""
        CREATE INDEX IF NOT EXISTS idx_history_last_read_at
        ON history_items(last_read_at DESC)
        """)
        db.execute("""
        CREATE INDEX IF NOT EXISTS idx_history_deleted_at
        ON history_items(deleted_at)
        """)
        db.commit()


def prune_history(db):
    rows = db.execute("""
        SELECT comic_id FROM history_items
        WHERE deleted_at IS NULL
        ORDER BY last_read_at DESC
        LIMIT -1 OFFSET ?
    """, (MAX_ITEMS,)).fetchall()
    ids = [row["comic_id"] for row in rows]
    if not ids:
        return
    placeholders = ",".join(["?"] * len(ids))
    db.execute(f"DELETE FROM history_items WHERE comic_id IN ({placeholders})", ids)


@app.on_event("startup")
def startup():
    init_db()


@app.get("/v1/health")
def health():
    return {"ok": True}


@app.get("/v1/history")
def get_history(
    limit: int = Query(default=200, ge=1, le=200),
    authorization: Optional[str] = Header(default=None),
):
    require_auth(authorization)
    with connect() as db:
        rows = db.execute("""
            SELECT * FROM history_items
            WHERE deleted_at IS NULL
            ORDER BY last_read_at DESC
            LIMIT ?
        """, (limit,)).fetchall()
    return {
        "items": [
            {
                "comicId": row["comic_id"],
                "title": row["title"],
                "author": row["author"],
                "thumbPath": row["thumb_path"],
                "thumbServer": row["thumb_server"],
                "lastReadAt": row["last_read_at"],
                "episodeOrder": row["episode_order"],
                "episodeTitle": row["episode_title"],
                "pageIndex": row["page_index"],
            }
            for row in rows
        ]
    }


@app.post("/v1/history/batch")
def upsert_history(
    payload: BatchIn,
    authorization: Optional[str] = Header(default=None),
):
    require_auth(authorization)
    updated_at = now_iso()

    with connect() as db:
        for item in payload.items[:MAX_ITEMS]:
            existing = db.execute(
                "SELECT last_read_at FROM history_items WHERE comic_id = ?",
                (item.comicId,),
            ).fetchone()
            if existing and existing["last_read_at"] > item.lastReadAt:
                continue

            db.execute("""
                INSERT INTO history_items (
                    comic_id, title, author, thumb_path, thumb_server,
                    last_read_at, episode_order, episode_title, page_index,
                    created_at, updated_at, deleted_at
                )
                VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, NULL)
                ON CONFLICT(comic_id) DO UPDATE SET
                    title = excluded.title,
                    author = excluded.author,
                    thumb_path = excluded.thumb_path,
                    thumb_server = excluded.thumb_server,
                    last_read_at = excluded.last_read_at,
                    episode_order = excluded.episode_order,
                    episode_title = excluded.episode_title,
                    page_index = excluded.page_index,
                    updated_at = excluded.updated_at,
                    deleted_at = NULL
            """, (
                item.comicId,
                item.title,
                item.author,
                item.thumbPath,
                item.thumbServer,
                item.lastReadAt,
                item.episodeOrder,
                item.episodeTitle,
                item.pageIndex,
                updated_at,
                updated_at,
            ))

        prune_history(db)
        db.commit()

    return {"ok": True}


@app.delete("/v1/history/{comic_id}")
def delete_history(
    comic_id: str,
    authorization: Optional[str] = Header(default=None),
):
    require_auth(authorization)
    ts = now_iso()
    with connect() as db:
        db.execute("""
            UPDATE history_items
            SET deleted_at = ?, updated_at = ?
            WHERE comic_id = ?
        """, (ts, ts, comic_id))
        db.commit()
    return {"ok": True}


@app.post("/v1/history/clear")
def clear_history(
    authorization: Optional[str] = Header(default=None),
):
    require_auth(authorization)
    ts = now_iso()
    with connect() as db:
        db.execute("""
            UPDATE history_items
            SET deleted_at = ?, updated_at = ?
            WHERE deleted_at IS NULL
        """, (ts, ts))
        db.commit()
    return {"ok": True}
EOF
```

### 6. Create nginx HTTPS Proxy

```bash
cat > nginx/default.conf <<'EOF'
server {
    listen 8443 ssl;
    server_name _;

    ssl_certificate /etc/nginx/certs/history.crt;
    ssl_certificate_key /etc/nginx/certs/history.key;

    ssl_protocols TLSv1.2 TLSv1.3;
    client_max_body_size 1m;

    location / {
        proxy_pass http://api:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Forwarded-Proto https;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
EOF
```

### 7. Create Docker Compose File

```bash
cat > docker-compose.yml <<'EOF'
services:
  api:
    build: ./app
    restart: unless-stopped
    env_file:
      - .env
    volumes:
      - ./data:/data

  nginx:
    image: nginx:1.27-alpine
    restart: unless-stopped
    depends_on:
      - api
    ports:
      - "8443:8443"
    volumes:
      - ./nginx/default.conf:/etc/nginx/conf.d/default.conf:ro
      - ./certs:/etc/nginx/certs:ro
EOF
```

### 8. Start Service

```bash
docker compose up -d --build
docker compose ps
```

Allow the HTTPS sync port. Do not expose port `8000`.

If this VPS already runs SSH, Hysteria2, or any other service, do not reset the
firewall and do not blindly enable UFW before allowing those existing ports. The
history sync service only needs TCP `8443`; keep your existing Hysteria2 port
and protocol rules unchanged.

```bash
ufw allow 8443/tcp
```

If UFW is already active, the command above is enough. If UFW is inactive and
you want to enable it, first allow SSH and your existing Hysteria2 port(s), then
run `ufw enable`.

### 10. Generate the App Certificate Pin

Run this on the VPS:

```bash
openssl x509 -in /opt/bika-history-sync/certs/history.crt -outform DER | openssl dgst -sha256 -binary | base64
```

Enter this value in the app settings as the certificate SHA-256 pin. The app also accepts 64-character lowercase/uppercase hex SHA-256 pins.

## App Configuration

In iOS or macOS settings, enable cloud history and enter:

```text
Server URL: https://<VPS_PUBLIC_IP>:8443
Token: <YOUR_LONG_TOKEN>
Certificate SHA256 pin: output from the pin command
```

These values are stored on the device. They are not committed to the repository.
