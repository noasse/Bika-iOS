# Cloud History Sync

This project supports an optional private cloud history sync service for iOS and macOS. It is off by default. If no server URL and token are configured in the app, reading history remains local only.

Do not commit your real VPS IP address, DuckDNS token, bearer token, certificate private key, or certificate pin to this repository. Enter runtime values in the iOS/macOS settings UI.

## Recommended Architecture

Use a real HTTPS hostname with a public CA certificate. This avoids iOS/macOS ATS failures caused by self-signed certificates.

```text
iOS/macOS app
  -> https://<YOUR_DUCKDNS_DOMAIN>
  -> Caddy with Let's Encrypt certificate
  -> FastAPI history service
  -> SQLite database capped at 200 items
```

Certificate SHA-256 pinning is optional:

- Leave the pin field empty when using DuckDNS + Caddy + Let's Encrypt.
- Fill the pin only when deliberately using a self-signed/private certificate.

The API stores one history row per comic and keeps the newest 200 non-deleted rows by `lastReadAt`.

## App Configuration

In iOS or macOS settings, enable cloud history and enter:

```text
Server URL: https://<YOUR_DUCKDNS_DOMAIN>
Token: <YOUR_HISTORY_SYNC_TOKEN>
Certificate SHA256 pin: leave empty for DuckDNS/Caddy
```

These values are stored on the device. They are not committed to the repository.

## Option A: Add Caddy In Front Of An Existing 8443 Service

Use this when the existing history service is already reachable on the VPS at `https://127.0.0.1:8443` with a self-signed certificate.

### 1. Preflight

Run these commands on the VPS. Stop if TCP `80` or TCP `443` is already used by another service.

```bash
ss -tulpn | grep -E ':(80|443|8443)\b' || true
docker ps --format 'table {{.Names}}\t{{.Ports}}\t{{.Status}}'
curl -k https://127.0.0.1:8443/v1/health
```

Expected health response:

```json
{"ok":true}
```

### 2. Install Caddy

```bash
apt update
apt install -y debian-keyring debian-archive-keyring apt-transport-https curl gnupg
curl -1sLf 'https://dl.cloudsmith.io/public/caddy/stable/gpg.key' | gpg --dearmor -o /usr/share/keyrings/caddy-stable-archive-keyring.gpg
curl -1sLf 'https://dl.cloudsmith.io/public/caddy/stable/debian.deb.txt' | tee /etc/apt/sources.list.d/caddy-stable.list
chmod o+r /usr/share/keyrings/caddy-stable-archive-keyring.gpg
chmod o+r /etc/apt/sources.list.d/caddy-stable.list
apt update
apt install -y caddy
```

### 3. Configure Caddy

Replace `<YOUR_DUCKDNS_DOMAIN>` with your DuckDNS hostname, for example `example.duckdns.org`.

```bash
cat > /etc/caddy/Caddyfile <<'EOF_CADDY'
<YOUR_DUCKDNS_DOMAIN> {
    encode zstd gzip

    reverse_proxy https://127.0.0.1:8443 {
        transport http {
            tls_insecure_skip_verify
        }
    }
}
EOF_CADDY

caddy validate --config /etc/caddy/Caddyfile
systemctl reload caddy
systemctl status caddy --no-pager
```

The `tls_insecure_skip_verify` setting is only between Caddy and the local self-signed 8443 service. The app still connects to Caddy with a public Let's Encrypt certificate.

### 4. Open Only Required TCP Ports

Do not reset the firewall and do not blindly enable UFW on a server that already runs SSH, Hysteria2, or other services.

```bash
ufw status
ufw allow 80/tcp
ufw allow 443/tcp
```

If UFW is inactive and you want to enable it, first allow SSH and your existing Hysteria2 port/protocol rules, then run `ufw enable` yourself.

### 5. Verify From Outside The VPS

```bash
curl https://<YOUR_DUCKDNS_DOMAIN>/v1/health
curl 'https://<YOUR_DUCKDNS_DOMAIN>/v1/history?limit=1' \
  -H 'Authorization: Bearer <YOUR_HISTORY_SYNC_TOKEN>'
```

Expected authorized history response:

```json
{"items":[]}
```

or an existing item list.

## Option B: New VPS Setup From Scratch

These commands assume Ubuntu 24.04 and root access. Replace every placeholder before running:

- `<YOUR_DUCKDNS_DOMAIN>`: your DuckDNS hostname.
- `<YOUR_HISTORY_SYNC_TOKEN>`: a long random token generated by `openssl rand -hex 32`.

### 1. Install Packages

```bash
apt update

systemctl enable --now docker
curl -1sLf 'https://dl.cloudsmith.io/public/caddy/stable/gpg.key' | gpg --dearmor -o /usr/share/keyrings/caddy-stable-archive-keyring.gpg
curl -1sLf 'https://dl.cloudsmith.io/public/caddy/stable/debian.deb.txt' | tee /etc/apt/sources.list.d/caddy-stable.list
```

### 2. Create Service Files

```bash
mkdir -p /opt/bika-history-sync/app

```

Generate a token:


Create `.env`:

```bash

```

Create Python dependencies:

```bash
EOF_REQ
```

Create the API service:

```bash
cat > app/main.py <<'EOF_APP'
import hmac
import os
import sqlite3
from datetime import datetime, timezone
from typing import Optional

from fastapi import FastAPI, Header, HTTPException, Query
from pydantic import BaseModel, Field

TOKEN = os.environ["HISTORY_SYNC_TOKEN"]
DB_PATH = os.environ.get("HISTORY_DB_PATH", "/data/history.sqlite3")
MAX_ITEMS = int(os.environ.get("HISTORY_MAX_ITEMS", "200"))

app = FastAPI(title="Bika History Sync")


class HistoryItemIn(BaseModel):
    comicId: str = Field(min_length=1)
    title: str = Field(min_length=1)
    author: Optional[str] = None
    thumbPath: Optional[str] = None
    thumbServer: Optional[str] = None
    lastReadAt: str
    episodeOrder: Optional[int] = None
    episodeTitle: Optional[str] = None
    pageIndex: Optional[int] = None


class BatchIn(BaseModel):
    items: list[HistoryItemIn]


def now_iso() -> str:
    return datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")


def connect():
    conn = sqlite3.connect(DB_PATH, timeout=5)
    conn.row_factory = sqlite3.Row
    return conn


def require_auth(authorization: Optional[str]):
    expected = f"Bearer {TOKEN}"
    if not authorization or not hmac.compare_digest(authorization, expected):
        raise HTTPException(status_code=401, detail="Unauthorized")


def init_db():
    with connect() as db:
        db.execute("""
        CREATE TABLE IF NOT EXISTS history_items (
            comic_id TEXT PRIMARY KEY,
            title TEXT NOT NULL,
            author TEXT,
            thumb_path TEXT,
            thumb_server TEXT,
            last_read_at TEXT NOT NULL,
            episode_order INTEGER,
            episode_title TEXT,
            page_index INTEGER,
            created_at TEXT NOT NULL,
            updated_at TEXT NOT NULL,
            deleted_at TEXT
        )
        """)
        db.execute("""
        CREATE INDEX IF NOT EXISTS idx_history_last_read_at
        ON history_items(last_read_at DESC)
        """)
        db.execute("""
        CREATE INDEX IF NOT EXISTS idx_history_deleted_at
        ON history_items(deleted_at)
        """)
        db.commit()


def prune_history(db):
    rows = db.execute("""
        SELECT comic_id FROM history_items
        WHERE deleted_at IS NULL
        ORDER BY last_read_at DESC
        LIMIT -1 OFFSET ?
    """, (MAX_ITEMS,)).fetchall()
    ids = [row["comic_id"] for row in rows]
    if not ids:
        return
    placeholders = ",".join(["?"] * len(ids))
    db.execute(f"DELETE FROM history_items WHERE comic_id IN ({placeholders})", ids)


@app.on_event("startup")
def startup():
    init_db()


@app.get("/v1/health")
def health():
    return {"ok": True}


@app.get("/v1/history")
def get_history(
    limit: int = Query(default=200, ge=1, le=200),
    authorization: Optional[str] = Header(default=None),
):
    require_auth(authorization)
    with connect() as db:
        rows = db.execute("""
            SELECT * FROM history_items
            WHERE deleted_at IS NULL
            ORDER BY last_read_at DESC
            LIMIT ?
        """, (limit,)).fetchall()
    return {
        "items": [
            {
                "comicId": row["comic_id"],
                "title": row["title"],
                "author": row["author"],
                "thumbPath": row["thumb_path"],
                "thumbServer": row["thumb_server"],
                "lastReadAt": row["last_read_at"],
                "episodeOrder": row["episode_order"],
                "episodeTitle": row["episode_title"],
                "pageIndex": row["page_index"],
            }
            for row in rows
        ]
    }


@app.post("/v1/history/batch")
def upsert_history(
    payload: BatchIn,
    authorization: Optional[str] = Header(default=None),
):
    require_auth(authorization)
    updated_at = now_iso()

    with connect() as db:
        for item in payload.items[:MAX_ITEMS]:
            existing = db.execute(
                "SELECT last_read_at FROM history_items WHERE comic_id = ?",
                (item.comicId,),
            ).fetchone()
            if existing and existing["last_read_at"] > item.lastReadAt:
                continue

            db.execute("""
                INSERT INTO history_items (
                    comic_id, title, author, thumb_path, thumb_server,
                    last_read_at, episode_order, episode_title, page_index,
                    created_at, updated_at, deleted_at
                )
                VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, NULL)
                ON CONFLICT(comic_id) DO UPDATE SET
                    title = excluded.title,
                    author = excluded.author,
                    thumb_path = excluded.thumb_path,
                    thumb_server = excluded.thumb_server,
                    last_read_at = excluded.last_read_at,
                    episode_order = excluded.episode_order,
                    episode_title = excluded.episode_title,
                    page_index = excluded.page_index,
                    updated_at = excluded.updated_at,
                    deleted_at = NULL
            """, (
                item.comicId,
                item.title,
                item.author,
                item.thumbPath,
                item.thumbServer,
                item.lastReadAt,
                item.episodeOrder,
                item.episodeTitle,
                item.pageIndex,
                updated_at,
                updated_at,
            ))

        prune_history(db)
        db.commit()

    return {"ok": True}


@app.delete("/v1/history/{comic_id}")
def delete_history(
    comic_id: str,
    authorization: Optional[str] = Header(default=None),
):
    require_auth(authorization)
    ts = now_iso()
    with connect() as db:
        db.execute("""
            UPDATE history_items
            SET deleted_at = ?, updated_at = ?
            WHERE comic_id = ?
        """, (ts, ts, comic_id))
        db.commit()
    return {"ok": True}


@app.post("/v1/history/clear")
def clear_history(
    authorization: Optional[str] = Header(default=None),
):
    require_auth(authorization)
    ts = now_iso()
    with connect() as db:
        db.execute("""
            UPDATE history_items
            SET deleted_at = ?, updated_at = ?
            WHERE deleted_at IS NULL
        """, (ts, ts))
        db.commit()
    return {"ok": True}
EOF_APP
```

Create the Dockerfile:

```bash
cat > app/Dockerfile <<'EOF_DOCKER'

```

Create Docker Compose. The API is bound to localhost only; Caddy is the public HTTPS entrypoint.

```bash
cat > docker-compose.yml <<'EOF_COMPOSE'

    restart: unless-stopped
    env_file:
      - .env
    ports:
      - "127.0.0.1:8000:8000"
    volumes:
      - ./data:/data
EOF_COMPOSE
```

## Legacy Self-Signed Notes

The old public-IP/self-signed setup is not recommended for iOS/macOS because ATS can reject the TLS handshake before app-level pinning is useful. If you intentionally use a private certificate, fill the app's optional certificate SHA-256 pin field. For DuckDNS + Caddy + Let's Encrypt, leave the pin empty.
